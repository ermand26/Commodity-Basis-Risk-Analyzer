# Commodity Basis Risk Analyzer

A quantitative finance tool built in Python to model, track, and analyze historical **basis risk** using real-time market data fetched via the Yahoo Finance API.

## Financial Context & Methodology
Basis risk occurs when an organization hedges an open physical position using financial futures, but the spread between the cash spot price and the futures contract fluctuates unexpectedly. 

Since physical spot commodity prices are often decentralized, this production-ready script proxies the "Spot vs. Futures" relationship by analyzing the spread between:
1. **Front-Month Futures Contract (`CL=F`)** - Serving as our Liquid Spot Proxy.
2. **Back-Month/Hedge Contract (`QM=F`)** - Serving as our Financial Hedge instrument.

The tool provides visual boundaries outlining a 1-Standard Deviation volatility zone to quantify this exact structural exposure over a 1-year historical window.

## Key Calculations Implemented
* **Historical Spread Basis**: $B_t = \text{Spot Proxy}_t - \text{Futures Hedge}_t$
* **Basis Risk Metric**: Standard Deviation ($\sigma$) of the historical basis spread.
* **Market Regimes**: Tracking transitions between Contango (negative basis) and Backwardation (positive basis).

## Installation & Setup
1. Clone this public repository.
2. Install the necessary quantitative analysis dependencies:
   ```bash
   pip install numpy pandas matplotlib yfinance
   ```
3. Run the analytical suite:
   ```bash
   python basis_risk_analyzer.py
   ```
